ACWW Web Save Editor is a work in progress Save Editor for Animal Crossing: Wild World made in JavaScript, HTML and CSS.

It is always recommended to have a backup of your savefile! Just in case something bad might happen!

If you are looking for an on console Save Editor, we suggest you'd look for WildEdit on a DS(i), or LeafEdit on a 3DS.

If you are interested into save research documentation, take a look here.
For a list of features, take a look here.
You can find the source code of ACWW Web Save Editor here.
