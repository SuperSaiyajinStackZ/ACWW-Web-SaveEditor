Welcome to ACWW Web Save Editor! Here you can read a few information about it.

ACWW Web Save Editor is a work in progress Save Editor for Animal Crossing: Wild World made in Javascript + HTML.

It is always recommended to have a backup of your savefile! Just in case something bad might happen!

If you are looking for an on console Save Editor, we suggest you'd look for WildEdit on a DS(i), or LeafEdit on a 3DS.

For a list of features, please take a look here.
